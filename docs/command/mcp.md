# iCal4j Model Context Protocol (MCP) Server

The iCal4j Model Context Protocol (MCP) Server is a reference implementation of the MCP specification,
which defines a protocol for managing and sharing calendar data.
## Features
- Implements the MCP specification for calendar data management
- Provides a RESTful API for interacting with calendar data
- Supports authentication and authorization for secure access
- Includes sample clients for testing and demonstration purposes
