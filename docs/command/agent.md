# iCal4j Agentic Services

This module provides agentic services for iCal4j, enabling autonomous agents to interact with calendar data. 
These services allow agents to create, read, update, and delete calendar events, as well as manage reminders 
and notifications.

## Features
- Create and manage calendar events
- Set reminders and notifications for events
- Support for recurring events
- Integration with external calendar services
- RESTful API for agent interaction
